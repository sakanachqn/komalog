# コマログ

駒を並べて自動戦闘 × ローグライクマップ進行のHTML5ゲーム。
全3幕の分岐マップを進み、最後に待つ魔王を討伐しよう。

## 起動

```
npm install
npm run dev   # http://localhost:5180
```

## 遊び方

- 分岐マップのノードを選んで進む（⚔️戦闘 / 💀エリート / 🛒ショップ / 🏕️休憩 / ❓イベント / 👑ボス）
- 戦闘前にユニットをクリック選択→マスをクリックで配置。戦闘は自動で進行
- 同じユニットを3体集めると自動で星アップ（★2、さらに3体で★3）
- 同系統のユニットを複数配置するとシナジー発動（戦士/射手/魔導士/守護者/暗殺者/狂戦士/僧侶/死霊/精霊の9系統・23ユニット）
- **アイテム**（8種）はユニットに1つ装備できる。通常戦闘で30%ドロップ、エリートで確定、ショップで5G
- **レリック**（10種）はラン全体に効く永続強化。エリート報酬の3択、ショップ（たまに14Gで入荷）、イベント「古代の祭壇」で入手
- プレイヤーHPが尽きる前に最上階の魔王を倒せば勝利

## 構成

- Vite + TypeScript、UIは素のDOM+CSS（フレームワーク・エンジン不使用）
- `src/battle.ts` — ティック制の自動戦闘シミュレーション（10tick/秒）
- `src/map.ts` — スレスパ風分岐マップ生成
- `src/state.ts` — ラン状態・ロースター・星アップ合成
- `src/data/units.ts` — 味方ユニット12種とシナジー定義
- `src/data/enemies.ts` — 敵とフロア別編成（フロア進行でスケール）
- `src/screens.ts` — 全画面のUI
